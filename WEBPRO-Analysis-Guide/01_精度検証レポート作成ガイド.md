## 第2部：分析項目と手法

### 2.1 基礎統計量（必須）

97件の誤差データから以下を算出する。

| 統計量 | 計算内容 | 相手への使い方 |
|--------|---------|---------------|
| **平均誤差（Mean Error）** | Σ(誤差) / N | バイアスの有無を示す |
| **中央値（Median）** | 誤差を昇順に並べた中央 | 外れ値の影響を除いた傾向 |
| **標準偏差 σ** | √(Σ(誤差-平均)² / (N-1)) | バラツキの大きさ |
| **最大値（Max）** | 最も大きい誤差 | 最悪ケース |
| **最小値（Min）** | 最も小さい誤差 | 反対方向の最悪ケース |
| **四分位範囲（IQR）** | Q3 - Q1 | σに代わるロバストな散布指標 |
| **1.28σの範囲** | 平均 ± 1.28σ | 相手の土俵で80%カバー率を確認 |
| **1.96σの範囲** | 平均 ± 1.96σ | 工学基準の95%カバー率 |
| **2.58σの範囲** | 平均 ± 2.58σ | 99%カバー率 |
| **範囲外件数（1.28σ）** | 実際に範囲外だった件数 | 理論20%との比較 |
| **範囲外件数（1.96σ）** | 実際に範囲外だった件数 | 理論5%との比較 |
| **RMSE** | √(Σ(誤差²) / N) | バイアスとバラツキの統合指標 |

> [!IMPORTANT]
> **相手が「1.28σ」を使ってくるなら、こちらは必ずRMSE（二乗平均平方根誤差）を提示せよ。** RMSEはバイアスとバラツキを同時に評価する指標であり、σだけでは見えない系統的偏りを暴露できる。RMSE² = バイアス² + σ² の関係がある。

### 2.2 分布分析（重要）

#### 2.2.1 正規性の検定

相手が「1.28σ」を使う前提は、誤差が**正規分布に従うこと**。これを検証する。

| 検定/指標 | 内容 | 判定基準 |
|----------|------|---------|
| **Shapiro-Wilk検定** | 正規性の統計検定 | p < 0.05 → 正規分布ではない |
| **歪度（Skewness）** | 分布の非対称性 | \|S\| > 1 → 明確に歪んでいる |
| **尖度（Kurtosis）** | 裾の重さ | \|K\| > 1 → 裾が重い（外れ値リスク大） |
| **Q-Qプロット** | 正規分布との目視比較 | 直線から逸脱 → 非正規 |

> [!WARNING]
> **正規分布でないなら「σで精度を語ること自体が統計的に不適切」と断言できる。** これは相手の論拠の根幹を崩す攻撃。

#### 2.2.2 ヒストグラム

- 横軸：誤差（BEIの差）
- 縦軸：件数
- 重ね描き：正規分布の理論曲線
- 追加表示：1.28σ、1.96σの範囲を色分け
- 追加表示：BEI判定反転ゾーンを赤でハイライト

#### 2.2.3 箱ひげ図（Box Plot）

- 全体の分布に加え、用途別・地域別の箱ひげ図を並べる
- 外れ値（1.5×IQR超）を点で明示

### 2.3 判定反転分析（最もインパクトが大きい）

#### 2.3.1 判定一致マトリクス（混同行列）

```
                    標準入力法の判定
                    適合(≤1.0)    不適合(>1.0)
モデル建物法  適合    TP(真陽性)    FP(偽陽性)←危険！
の判定      不適合   FN(偽陰性)    TN(真陰性)
```

| 指標 | 計算式 | 意味 |
|------|--------|------|
| **偽陽性率（FPR）** | FP / (FP+TN) | 本来不適合なのに適合と判定する確率 |
| **偽陰性率（FNR）** | FN / (FN+TP) | 本来適合なのに不適合と判定する確率 |
| **判定不一致率** | (FP+FN) / N | 全体での判定誤り率 |

> [!CAUTION]
> **偽陽性（FP）: 本来不適合の建物が適合と判定される** → 法令違反の建物が見逃される → これが最大のリスク。1件でもあれば致命的な反論材料になる。

#### 2.4.4 統計的検出力（Power Analysis）

**「差がないと言えるためにも十分なサンプルが必要」** を数学的に示す。

BEIの差 δ=0.03（3%相当）を検出するための必要サンプル数：

```
前提条件：
  α = 0.05（有意水準 5%）
  Power = 0.80（検出力 80%）
  δ = 0.03（検出したい差）
  σ = 0.05（仮定）

必要N = (Zα/2 + Zβ)² × 2σ² / δ²
     = (1.96 + 0.84)² × 2 × 0.05² / 0.03²
     = 7.84 × 0.005 / 0.0009
     ≒ 43.6 → 44件（1群あたり）

用途別に検出力を確保するには：
  8用途 × 44件 = 352件が最低必要
```

> [!NOTE]
> σの実際の値が0.05より大きければ必要サンプル数はさらに増加する。

#### 2.5.2 Bland-Altman プロット（重要！）

2手法の一致度を評価する国際標準的な手法。

- 横軸：(標準入力BEI + モデル建物法BEI) / 2（平均）
- 縦軸：モデル建物法BEI − 標準入力法BEI（差）
- 水平線：平均差（バイアス）
- 上下限線：平均差 ± 1.96σ（95%一致限界）

> [!TIP]
> Bland-Altmanプロットは医学・工学で**2つの測定法の一致度を評価する標準手法**。これを出すと学術的な信頼性が格段に上がる。相手がσだけで語ってくるレベルなら、これを出した時点で議論の格が違うことを見せつけられる。

### 2.6 回帰分析（原因の特定）

#### 2.6.1 単回帰
誤差を目的変数として、各パラメータとの関係を個別に確認。

```
誤差 = a × (延床面積) + b      → 規模による影響
誤差 = a × (窓面積比率) + b    → 外皮仕様による影響
誤差 = a × (空調COP) + b      → 空調効率による影響
```

#### 2.6.2 重回帰
```
誤差 = β₀ + β₁(用途) + β₂(地域) + β₃(規模) + β₄(空調方式) + ... + ε
```
- 自由度調整済みR²で説明力を確認
- 各βの有意性（p値）で寄与度を特定
- **VIF（分散拡大係数）** で多重共線性をチェック

> [!WARNING]
> N=97で15変数の重回帰は自由度が不足する。ステップワイズ法で変数選択するか、主要5変数程度に絞る必要がある。この事実自体が「N=97では精度の原因分析すら困難」という主張の根拠になる。
