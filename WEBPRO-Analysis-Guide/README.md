# WEBPRO Analysis Guide

WEBPRO（建築物省エネ法に基づくエネルギー消費性能計算プログラム）に関する分析ガイド集。

## 📂 フォルダ構成

```
WEBPRO-Analysis-Guide/
├── README.md                              # 本ファイル
├── 01_精度検証レポート作成ガイド.md          # メインガイド：簡易入力の精度問題を論証するための完全マニュアル
├── 02_データ収集テンプレート.md              # 分析に必要なデータ収集用テンプレート
└── 03_反論対策クイックリファレンス.md         # ステークホルダーへの反論対策一覧
```

## 🎯 目的

**モデル建物法（簡易入力）の精度リスク** をデータに基づき定量的に論証し、標準入力法の採用を推奨するための資料群。

## 📋 主な内容

1. **精度検証レポート作成ガイド** — N=97件のデータを用いた分析手法の完全マニュアル
2. **データ収集テンプレート** — BEI値、設計パラメータ等の収集用テンプレート
3. **反論対策クイックリファレンス** — ステークホルダーからの反論への切り返し集

## ⚡ クイックスタート

1. `02_データ収集テンプレート.md` でデータ項目を確認
2. `01_精度検証レポート作成ガイド.md` に沿って分析を実施
3. `03_反論対策クイックリファレンス.md` でプレゼン準備
