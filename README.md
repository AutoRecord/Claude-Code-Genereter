# Claude Code Generator - 生産性ツールコレクション

ブラウザだけで動作する、スタンドアロンの生産性ツールコレクションです。
ビルド不要・インストール不要で、HTML ファイルを開くだけですぐに使えます。

## ツール一覧

### マインドマップ

Markdown をリアルタイムでマインドマップに変換するエディタです。

- 左ペインに Markdown を入力すると、右ペインにマインドマップが即座に描画
- Monaco Editor によるシンタックスハイライト付きエディタ
- ファイルの読み込み・保存（Ctrl+O / Ctrl+S / Ctrl+Shift+S）
- ダークテーマ UI

**使用ライブラリ:** Monaco Editor, Markmap, D3.js

### マンダラチャート

大谷翔平選手でも知られる 9×9 のマンダラチャート（目標設定シート）ツールです。

- 中央セルにメインの目標を入力すると、周囲のブロックに自動反映
- 5 種類のカラープリセット（クラシック / パステル / ビビッド / オーシャン / モノクロ）
- ブロックごとの色カスタマイズ、フォントサイズ調整
- PNG / PDF エクスポート
- LocalStorage による自動保存、複数チャートの管理
- キーボードナビゲーション（Tab / Ctrl+矢印キー）

**使用ライブラリ:** html2canvas, jsPDF

## 使い方

1. このリポジトリをクローンまたはダウンロード
2. 各フォルダ内の HTML ファイルをブラウザで開く

```
マインドマップ/マインドマップ.html
マンダラチャート/マンダラチャート.html
```

> ライブラリは CDN から読み込むため、インターネット接続が必要です。

## 動作環境

- Chrome / Edge / Firefox / Safari（最新版推奨）
- JavaScript 有効
- インターネット接続（CDN からのライブラリ読み込み）

## ディレクトリ構成

```
Claude-Code-Genereter/
├── README.md
├── マインドマップ/
│   └── マインドマップ.html
└── マンダラチャート/
    └── マンダラチャート.html
```
