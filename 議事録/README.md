# 📋 議事録テンプレート

## ファイル一覧

| ファイル | 用途 | 使い方 |
|---------|------|--------|
| `議事録テンプレート_走り書き.txt` | 会議中のリアルタイムメモ | テキストエディタで開いてそのまま入力 |
| `議事録テンプレート_走り書き_記入例.txt` | ↑の書き方サンプル | 参考用。こう書くとイメージ |
| `議事録テンプレート_OneNote.htm` | 清書・共有用（OneNote貼付け） | ブラウザで開く→全選択コピー→OneNoteに貼付け |
| `議事録運用ガイド.md` | 運用ルール・コツ集 | チームで共有して標準化に活用 |

---

## 🚀 おすすめ運用フロー

### Step 1：会議前
- `走り書き.txt` をコピーして新規ファイルを作成（ファイル名例：`2026-03-27_下半期報告.txt`）
- アジェンダと出席者を事前に記入
- 前回の議事録からTODOをコピペして「前回アクションアイテムの確認」に転記

### Step 2：会議中
- `名前>` 形式で発言をガシガシ記録
- 重要発言には ★ マークを付ける
- テーブルは気にせず、テキストで走り書き

### Step 3：会議後（当日中 or 翌営業日まで）
- 走り書きを整理
- 決定事項・TODO・保留事項を確認
- 必要に応じてOneNoteテンプレートに清書
- 関係者に共有（合意確認の期限を設定）

---

## 💡 走り書きテンプレートの記法ルール

```
名前>  ← 発言記録。名前のあとに > をつけるだけ
★     ← 重要な発言につけるマーク
→     ← 決定事項の決定者・合意者への矢印
/      ← 区切り（依頼者：名前 / 期限：日付）
```

---

## 🔑 テンプレートの設計思想

このテンプレートは **「言った言わない問題」の防止** を最重要目的として設計しています。

1. **全セクションに発言者を記録** → 誰が何を言ったかを明確化
2. **決定事項に「合意者」列** → 誰が合意したかを証跡として残す
3. **アクションアイテムに「依頼者」列** → 誰が依頼したかを記録
4. **合意確認の注意書き** → 期限内に異議がなければ合意済みとみなす
5. **Teams録画リンク** → 発言の裏付け証拠
6. **前回TODO確認セクション** → 宿題の抜け漏れ防止
